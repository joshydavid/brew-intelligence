## Getting Started

1. Clone the project

   ```
   git clone https://github.com/joshydavid/brew-intelligence.git
   ```

2. Install dependencies

   ```
   bun install
   ```

3. Set up environment variables

   `cp .env.sample .env`

4. Run the project in development environment

   ```
   bun run dev
   ```
